HTML & CSS

HTML è un linguaggio che permette di strutturare i contenuti di una pagina web, come testi, immagini e link. CSS è utilizzato per definire lo stile e l'aspetto visivo della pagina, controllando colori, font e layout. Insieme, HTML e CSS sono fondamentali per creare e personalizzare l'aspetto delle pagine web.

JavaScript

JavaScript è un linguaggio di programmazione che rende interattive le pagine web. Permette di aggiungere dinamismo, effetti speciali e interazioni in tempo reale senza ricaricare la pagina. È essenziale per creare applicazioni web più moderne e coinvolgenti, interagendo con l'utente e i dati in modo fluido e rapido.

Bootstrap

Bootstrap è un framework open source che facilita lo sviluppo di siti web moderni e responsive. Fornisce strumenti predefiniti, come layout, componenti e stili CSS, per creare pagine web ottimizzate per diversi dispositivi. Con Bootstrap, è possibile realizzare design coerenti e professionali.
